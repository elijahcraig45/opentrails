#### Option A: GitHub Branch Protection (Recommended)
1. Go to Settings → Branches → Add rule
2. Target: `main`
3. Check:
   - ✅ Require pull request reviews (1)
   - ✅ Require status checks to pass
   - ✅ Require branches to be up to date
4. Click **Create**

This prevents accidental pushes and ensures quality.

#### Option B: Vercel Auto-Deploy
1. Go to https://vercel.com/dashboard
2. Click **Add New Project** → **Import Git Repository**
3. Authorize GitHub
4. Select `opentrails`
5. Framework: **Expo**
6. Root Directory: **app**
7. Add Environment Variable:
   - Name: `REACT_APP_API_URL`
   - Value: `https://opentrails-api-542596148138.us-central1.run.app/api`
8. Click **Deploy**

After ~5 minutes, your app will be live at:
```
https://opentrails-xxxxx.vercel.app
```
